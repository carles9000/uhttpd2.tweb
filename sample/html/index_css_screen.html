When should we use this technique

Cuando usas el mismo css en varios prgs, lo puedes grabar en un fichero, p.e. en files/css/myfile.css y luego cargarlo desde tu prg de esta manera oWeb:AddCss( 'files/css/mybrw.css' )

En este ejemplo hay 3 módulos que usan un browse. Solo tienes un mismo fichero css único para estos 3 o 50. El mantenimiento es mas fácil.

When you use the same css in several prgs, you can save it in a file, e.g. in files/css/myfile.css and then load it from your prg like this oWeb:AddCss( 'files/css/mybrw.css' )

In this example there are 3 modules that use a browse. you only have one Single CSS file for these 3 or 50. Maintenance is easier.
